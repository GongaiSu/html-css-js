黑龙江省有这样一个城市，名字叫做加格达奇，是大兴安岭地区的首府。然而，这座城市却坐落在内蒙古自治区境内，四周与黑龙江根本不接壤。为了这块飞地，黑龙江每年要给内蒙古自治区上交700万！加格达奇因此成为了中国最神奇的城市。1969年内蒙古境内的大兴安岭林区连同当时的呼伦贝尔盟一道划归黑龙江省管理，直到1979年呼伦贝尔市重新划归内蒙古，但是在呼伦贝尔市境内的加格达齐这个“孩子”已经被养大，并且是大兴安岭地区的核心，所以内蒙古想要再“要”回来，已经没什么可能了。后来内蒙古自治区还专门给中央写报告，要求收回加格达奇，但黑龙江也不放弃到嘴的肥肉。后中央和国务院明确批示加格达奇归黑龙江领导，原属地权不变，税收归内蒙古。但是，黑龙江省也要付出代价，黑龙江省政府每年向内蒙古自治区政府支付700万元。
